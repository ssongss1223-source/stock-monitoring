**📈 주식 신호 알림 시스템**

전략 설계 브레인스토밍 문서

버전 1.0  |  2025년 작성

# **1. 시스템 개요**

본 시스템은 완전 자동매매가 아닌, 종목 추천 및 매도 타이밍 알림에 집중한 의사결정 지원 시스템입니다.

| **항목** | **내용** |
| --- | --- |
| 목적 | 종목 추천 + 매도 신호 알림 (매수는 사람이 직접) |
| 대상 시장 | 국내 주식 (코스피, 코스닥) |
| 매매 스타일 | 스윙 트레이딩 (3일 ~ 3개월) |
| 실행 빈도 | 매일 장 마감 후 1회 (16:00) |
| 알림 수단 | 텔레그램 봇 |
| 개발 환경 | Python + Miniforge (Windows) |
| 인프라 | 개인 PC 또는 GCP 무료 티어 |

# **2. 핵심 철학**

복잡한 시스템보다 검증된 기법을 정확하게 구현하는 것이 핵심입니다.

| **원칙** | **설명** |
| --- | --- |
| 손익비 우선 | 승률보다 손익비가 중요. 승률 40%여도 손익비 1:3이면 장기 수익 |
| 앙상블 스코어링 | 단일 지표 맹신 금지. 여러 지표 점수 합산으로 오신호 감소 |
| 장세 필터 | 하락장에서 신규 매수 추천 차단. 시장이 나쁘면 무조건 패스 |
| 사람이 최종 판단 | 시스템은 신호만. 매수 실행은 사람이 직접 판단 |
| 단계적 고도화 | 기본 기능 먼저 완성 후 외부 요인 등 점진적 추가 |

# **3. 전체 시스템 구조**

아래 5단계 레이어로 구성됩니다.

[1단계] 거시 장세 판단

  코스피/코스닥 1개월·3개월·6개월 추세 분석

  상승장 / 횡보장 / 하락장 판별

  하락장 → 신규 매수 추천 전면 차단

[2단계] 섹터/업종 분석

  종목이 속한 업종 추세 확인

  업종 아웃퍼폼 여부 판단

[3단계] 종목 개별 분석 (앙상블)

  다중 이동평균 + 일목균형표 + RSI + MACD + 볼린저밴드

  패턴 인식: 컵앤핸들 / 밥그릇 / 삼각수렴 / 볼린저수축

  추세선/채널 + 피보나치 + 매물대 분석

[4단계] 거래량 분석 (핵심 엔진)

  거래량 폭발 예측 (터지기 전 징조 감지)

  OBV, 거래량 이동평균 크로스, 매집 패턴

[5단계] 외부 요인 필터

  뉴스 감성분석, 공시 감지, 환율/금리 연동

[최종 출력] 종합 스코어 → 등급(S/A/B) → 텔레그램 알림

# **7. 패턴 인식 기법**

## **7-1. 컵앤핸들 (Cup ****&**** Handle)**

- 기간: 수주 ~ 수개월

- 손익비: 1:3 ~ 1:5

- 매수 타점: 핸들 고점 돌파 + 거래량 폭발

- 손절: 핸들 저점 아래

- 목표: 컵 깊이만큼 위

## **7-2. 밥그릇 (Rounding Bottom)**

- 기간: 수개월 (더 긴 추세)

- 손익비: 1:4 ~ 1:6

- 매수 타점: 네크라인 돌파 시

- 핵심: 바닥에서 거래량 서서히 증가

## **7-3. 볼린저밴드 수축→팽창**

- 밴드 수축 = 에너지 응축, 거래량 감소

- 밴드 팽창 = 방향성 폭발

- 컵앤핸들 핸들 구간과 겹치면 S급 신호

## **7-4. 삼각수렴 돌파**

- 고점은 낮아지고 저점은 높아지며 거래량 감소

- 돌파 시 거래량 폭발 + 강한 상승

# **8. 지지/저항 분석 (타점 정교화)**

여러 지지/저항이 겹치는 구간일수록 손절은 짧고 목표는 멀어져 손익비가 극대화됩니다.

## **8-1. 지지/저항 레이어**

| **기법** | **종류** | **특징** |
| --- | --- | --- |
| 매물대 (Volume Profile) | 정적 지지/저항 | POC, VAH, VAL 기반. 목표가 설정에 활용 |
| 일목 구름대 | 동적 지지/저항 | 두께가 강도. 구름대 이탈 = 손절 기준 |
| 추세선/채널 | 기울기 지지/저항 | 고점-고점, 저점-저점 연결 |
| 피보나치 되돌림 | 비율 지지/저항 | 61.8% 황금비율이 가장 강한 지지 |
| 수평 지지/저항 | 가격 지지/저항 | 전고점, 전저점, 52주 고저 |

## **8-2. 겹침 구간 활용 예시**

지지 겹침 구간: 74,000 ~ 74,500원

  ✅ 상승 추세선 지지

  ✅ 피보나치 61.8% 레벨

  ✅ 일목 구름대 상단

  ✅ 강한 매물대 POC

다음 저항 겹침: 85,000원

  ✅ 하락 추세선 저항

  ✅ 전고점

  ✅ 매물대 VAH

매수: 74,200원 / 손절: 73,500원 (-0.9%) / 목표: 85,000원 (+14.5%)

→ 손익비 약 1:16 🔥

# **13. 백테스트 계획**

## **테스트 기간**

- 설계 기간: 2024년 이전 데이터로 전략 수립

- 검증 기간: 2025년 4월 ~ 2026년 4월 (순수 검증용)

## **이 기간이 좋은 이유**

| **기간** | **시장 특성** |
| --- | --- |
| 2025년 4월 | 트럼프 관세 쇼크 → 급락장 |
| 2025년 중반 | 반등 구간 |
| 2025년 하반기 | 변동성 장세 |
| 2026년 초 | 회복 구간 |

상승/하락/횡보/급변동이 모두 포함된 이상적인 검증 구간입니다.

## **검증 항목**

- 하락장 감지 정확도 (2025년 4월 폭락 시 매수 중단했는가)

- 반등 초입 포착 (거래량 신호 잡았는가)

- 매도 타점 정확도 (고점 근처에서 신호 나왔는가)

- 오신호 비율 (전체 추천 중 실패 비율)

- 평균 손익비 달성 여부

## **⚠️ 과적합 방지 원칙**

- 검증 기간 데이터로 파라미터 조정 절대 금지
