## 이번 세션(2026-05-05 세션8) 변경 내용

| 파일 | 변경 내용 |
|---|---|

**패턴 변경 배경:**
- `rounding_bottom`: U자형 평균값만 비교 — 하락 채널 유무, 돌파 여부 무관 → 오탐 높음
- `falling_box_breakout`: 하락 채널 존재 → 상단 돌파 → 완만한 조정 시퀀스 검증 → 손익비 명확

**`falling_box_breakout` 알고리즘 요약:**
1. 직전 40일 고점·저점 기울기 모두 음수 + 낙폭 10% 이상 (하락 채널 확인)
2. 최근 25일 내 채널 상단 추세선 위로 돌파 확인
3. 돌파 후 조정일수 ≥ 상승일수 × 1.5 (건강한 조정 각도)
4. 조정 폭 50% 미만 + 채널 상단선 ±5% 위에서 유지

**패턴 우선순위:** cup_handle → falling_box_breakout → triangle_convergence → bb_squeeze

*마지막 업데이트: 2026-05-05 세션8 — 패턴 rounding_bottom → falling_box_breakout 교체*
