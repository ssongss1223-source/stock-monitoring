## 새 세션 기록 방법
1. 현재 달의 파일을 연다. 없으면 `docs/work-log-YYYY-MM.md`로 새로 만든다.
2. 파일 **맨 위** (헤더 바로 아래)에 새 세션을 추가한다 (최신이 위).
3. 형식:

```
## YYYY-MM-DD 세션 N — 한 줄 요약
- 작업:
- 변경 사항:
- 관련 파일:
- 메모:
- 다음 아이디어:
```
